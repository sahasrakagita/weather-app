# 🌦️ Weather App

An interactive and animated Weather Application built using JavaScript and OpenWeatherMap API.  
This app allows users to search for real-time weather information of any city.

---

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Displays temperature in Celsius
- 🌤️ Shows weather condition (Clouds, Rain, Clear, etc.)
- 💧 Displays humidity
- 🎨 Dynamic background changes based on weather
- ✨ Smooth animations and modern UI
- ⌨️ Press Enter to search

---

## 🛠️ Tech Stack

- HTML
- CSS (with animations)
- JavaScript (ES6)
- API integration

---

## 🔗 API Used

- OpenWeatherMap API


---

## ⚙️ How to Run

1. Download or clone this repository
2. Open the project folder in VS Code
3. Add your API key in `script.js`
4. Open `index.html` in browser
